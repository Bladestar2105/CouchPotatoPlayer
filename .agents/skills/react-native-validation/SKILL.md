# React Native Validation Skill

## Purpose
Use this skill when validating mobile-related changes.

## Checks
- type safety
- lint
- tests
- build sanity

## Watch for
- invalid hooks usage
- navigation regressions
- async cleanup problems
- state update loops
- platform-specific issues

## Commands
- npm run typecheck
- npm run lint
- npm test
- npm run build

## Output
- checks run
- failures
- impact
